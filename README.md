## 一个空的composer的结果及其文件的解说

* ├── .editorconfig      # 编辑器配置文件，比如缩进大小、换行模式等
* ├── .gitattributes     # git 配置文件，可以设计导出时忽略文件等
* ├── .gitignore         # git 忽略文件配置列表
* ├── .php_cs            # PHP-CS-Fixer 配置文件
* ├── README.md    
* ├── composer.json
* ├── phpunit.xml.dist
* ├── src
* │   └── .gitkeep
* └── tests
*     └── .gitkeep

######	将文件重命名为，** .editorconfig **，** .gitattributes ** ，** .gitignore ** ，** .php_cs **
